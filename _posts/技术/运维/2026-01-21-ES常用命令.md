# 1.常用命令

ES使用 REST API进行交互，一些常用命令可以参考[Elasticsearch中文文档](https://elasticsearch.bookhub.tech/rest_apis/)
下面描述一些没有收录的命令


**创建索引**
```bash
PUT /indexName
```
**查看ES所有索引及其状态**
```bash
GET /_cat/indices?v
```
